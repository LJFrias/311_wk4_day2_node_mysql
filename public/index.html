This backend express/sql appincludes the following routes:

GET /users

Will provide a list of all the users from the database with and id, first name, last name, address, city, county, state, zip, primary phone, secondary phone, and email

GET /users/:id

Will provide you with a single user by id. Provided they exist

POST /users

Will allow you to add an entry for a new user that will be assigned an ID and returned to you as long as you provide the first name, last name, address, city, county, state, zip, primary phone, secondary phone, and email.

PUT /users/:id

Will allow you to update a specific user's first and last name with the new information specified in the body of the request. Provided they exist.

DELETE /users/first_name

Will allow you to delete a user or users from the database by specifyig the first name in the body of the request

All routes tested in Postman
